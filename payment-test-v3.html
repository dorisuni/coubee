🚀 PortOne V2 결제 테스트 (로그인 기능)

⚙️ V2 설정 가이드

1. PortOne 콘솔에서 V2 계정 생성

2. Store ID와 API Secret 발급

3. 아래 설정 파일에 실제 값 입력:

application.yml:
portone:
  v2:
    store-id: "실제_스토어_ID"
    api-secret: "실제_API_시크릿"

📱 2단계: V2 결제 진행

🧪 3단계: API 테스트 도구들
